# Footpath walkability assesment using images

footpath walkability assesment using images in guwahati ,classify the fothpath quality as safe,moderate,unsafe

## Prerequisites

Before you begin, ensure you have the following installed:

- Python 3.x
- pip (Python package installer)
- Model File: Place the model file (walkability_model.pt) in the project directory.

## Installation

Follow these steps to set up your virtual environment and install the required dependencies:

### Step 1: Create a Virtual Environment

```bash
$ python3 -m venv .venv
```

### Step 2: Activate the Virtual Environment

On UNIX or MacOS:

```bash
$ source .venv/bin/activate
```

On Windows:

```bash
$ .venv\Scripts\activate
```

### Step 3: Install Required Packages

Run the following command to install Flask and other dependencies:

```bash
$ pip install flask torch torchvision pillow
```

## Running the Application

### Step 1: Start the Application

Execute the following command to run the app:

```bash
$ python app.py
```

### Step 2: Open the Application in Your Browser

Navigate to the following URL in your web browser:

```
http://127.0.0.1:5000
```
